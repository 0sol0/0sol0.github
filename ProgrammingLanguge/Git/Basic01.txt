버전관리 : 이 프로젝트가 언제, 어디서, 어떤 것을 했는지 저장하고 관리하는 것
commit : 현재 이 프로젝트의 정보를 저장하는 것
repository : Git으로 관리되는 Project
	로컬 repo : 내 컴퓨터에 저장되어있는 repo
	원격 repo : 다른 곳에 저장되어있 repo
tracking(추적) : 로컬 repo에서 원격 repo를 추적하는 것

push(푸쉬) : 로컬 repo에 있는 것을 원격 repo로 보내는 것
pull(풀) : 원격 repo에 있는 것은 로컬 repo로 가져오는 것
clone(복제) : 로컬 repo에 아무것도 없을 때, 원격 repo에 있는 것을 로컬 repo에 저장하는 것
cofilct(충돌) : 로컬 repo와 원격 repo에서 같은 파일을 수정할 때 생기는 것
