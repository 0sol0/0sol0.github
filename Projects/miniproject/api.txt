멤버 카드 버튼 - 추가버튼 -> 입력장 - 저장버튼
        |
    수정버튼 -> 입력창 - 저장버튼

처음 시작할 때 리로드
추가 버튼 클릭 -> 입력창 리셋 -> 입력창 열림
입력창에 데이터 입력 -> 저장 버튼 클릭 -> 데이터 저장 -> 입력창 닫힘
멤버 카드 클릭 -> 소개창 리로드 -> 소개창 열림
수정 버튼 클릭 -> 입력창 리셋 -> 입력창 열림
입력창에 데이터 입력 -> 저장 버튼 클릭 -> 비밀번호 확인 -> 데이터 저장 -> 입력창 닫힘
멤버 카드 클릭 -> 소개창 닫힘



1. 추가 버튼 클릭 하면 입력창이 뜸
2. 입력창에 데이터 입력
3. 비밀번호 입력 후 저장
4. 저장 버튼 클릭하면 닫힘
- 입력된 값 리셋 -
- 비밀번호 및 입력창에 쓴 데이터 저장 -
- 입력창에 쓴 데이터를 보내줌 -
5. 멤버카드 클릭하면 멤버 소개 뜸
6. 한번 더 클릭하면 멤버 소개가 닫힘
7. 수정 버튼 클릭하면 입력창이 뜸
8. 입력창에 데이터 입력
9. 비밀번호 입력 후 저장
10. 저장 버튼 클릭하면 닫힘
- 입력값 리셋 -
- 비밀번호 일치 확인 및 입력창에 쓴 데이터 저장 -
- 입력창에 쓴 데이터를 보내줌 -
